User
	URL: https://github.com/$1
	Examples:
		Twitter -> twitter
		Mukunda Modell -> 20after4
		jayme-github -> jayme-github
		Riccardo Coccioli -> volans-
Repository
	URL: https://github.com/$1
	Examples:
		Twemoji -> twitter/twemoji
Tree
	URL: https://github.com/$userid/$repoid/tree/$id
	Examples:
		master -> master
		gh-pages -> gh-pages
Marketplace
	URL: https://github.com/marketplace/$id
	Examples:
		Flaptastic -> flaptastic
		Jira Software + GitHub -> jira-software-github
		HTML/CSS to Image -> html-css-to-image
